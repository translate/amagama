About amaGama

amaGama is a web service written in Python implementing a large-scale translation memory on top of the PostgreSQL database server. A translation memory is a database of previous translations which can be searched to find good matches to new strings.

amaGama can be used in conjunction with the Virtaal and Pootle translation tools.
